# Amazon MemoryDB

Amazon MemoryDB is a durable, in-memory database service that delivers ultra-fast performance. It is purpose-built for modern applications with microservices architectures. MemoryDB is compatible with Redis and Memcached, giving you the flexibility to use the same popular and open source data structures, APIs, and commands that they already use today. With MemoryDB, all of your data is stored in memory, which enables you to achieve microsecond reads and single-digit millisecond writes with high throughput. It also durably stores data across multiple Availability Zones using a distributed transactional log.

## APIs

- **Amazon MemoryDB API** - The Amazon MemoryDB API provides programmatic access to create and manage MemoryDB clusters, parameter groups, subnet groups, snapshots, users, and ACLs for Redis-compatible in-memory databases.
